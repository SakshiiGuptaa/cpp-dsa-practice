Sorting Algorithms ->

Selection sort -> TC =O(N^2)
Insertion sort -> TC = O(N^2) at worst and average case, O(N) at best case
Bubble sort -> TC = O(N^2) at worst and average case, O(N) at best case
Merge sort -> divide and merge -> TC = O(NlogN), logN for merge sort recursive approach and O(N) for merging even at worst condition, SC = O(N)
Quick sort -> divide and conquer -> TC = O(NlogN), logN for sort based on partition recursive approach and O(N) for running the partition logic, SC = O(1)
