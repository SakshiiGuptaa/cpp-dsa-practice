# Arrays

This folder contains array-based problems from the Striver A to Z DSA Sheet.

Topics covered:
- Largest Element
- Second Largest Element
- Check if array is sorted or not
- Remove duplicates from array
- Rotate array by one place (left, right)
- Rotate array by k places (left, right)
- Move all zeroes at end of array
- Linear search
- Union of two sorted arrays
- Intersection of two sorted arrays
- Missing number
- Maximum consecutive 1's
- Find number that appears once and others twice
